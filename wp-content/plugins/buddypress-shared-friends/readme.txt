Shows shared friends between the loggedin user and other users in a buddypress community.

== Description ==

The Plugin shows a list of users (with their gravatars) that are friends both with the loggedin and with the displayed user. The list of shared friends can be displayed at the bottom of the profil page or as a widget.

Deutsch:
Das Plugin zeigt User an, die sowohl mit dem eingeloggten, als auch mit dem angezeigten Mitglied befreundet sind (gemeinsame Freunde). Die Liste der gemeinsamen Freunde kann unter den Profilinformationen, oder als Widget angezeigt werden.

More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Upload the sxss Buddypress Shared Friends plugin to your blog, activate it, go to 'Settings > sxss Shared Friends'. After this you can go to 'Appearance > Widgets' and add the 'Shared Friends' Widget.
